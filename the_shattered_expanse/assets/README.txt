Required Sprite Files and Specifications:

Character Sprites:
- player.png (32x32)
  - Main player character
  - Should be recognizable from top-down view
  - Recommended: Red/brown color scheme

Enemy Sprites:
- scout.png (25x25)
  - Fast, light enemy
  - Orange color scheme
  - Triangle or arrow shape recommended
  
- warrior.png (32x32)
  - Standard enemy
  - Red color scheme
  - Square or humanoid shape
  
- heavy.png (40x40)
  - Tank-type enemy
  - Dark red color scheme
  - Hexagonal or bulky shape
  
- ranged.png (28x28)
  - Ranged attacker
  - Purple color scheme
  - Diamond or star shape
  
- boss.png (50x50)
  - Boss enemy
  - Gold/yellow color scheme
  - Large, intimidating design

Building Sprites:
- wall.png (40x40)
  - Basic defensive structure
  - Gray/stone texture
  - Simple solid barrier design
  
- turret.png (40x40)
  - Advanced defensive structure
  - Metallic with weapon features
  - Should have visible gun/cannon
  
- storage.png (60x60)
  - Resource storage building
  - Brown/wooden appearance
  - Container/warehouse style
  
- workshop.png (80x80)
  - Crafting/production building
  - Gray/industrial look
  - Should show workbenches/tools
  
- collector.png (40x40)
  - Resource gathering structure
  - Green/technological appearance
  - Moving parts suggested
  
- generator.png (50x50)
  - Power production building
  - Orange/energetic appearance
  - Visible power effects

Sprite Requirements:
1. All sprites should be PNG format with transparency
2. Maintain exact pixel dimensions for proper scaling
3. Use consistent perspective (top-down view)
4. Include basic animations frames if possible
5. Clear silhouettes for easy recognition
6. Consistent lighting direction
7. Color schemes should match faction/type

Directory Structure:
/assets/
  - player.png
  - enemies/
    - scout.png
    - warrior.png
    - heavy.png
    - ranged.png
    - boss.png
  - buildings/
    - wall.png
    - turret.png
    - storage.png
    - workshop.png
    - collector.png
    - generator.png

If sprite files are missing, the game will use colored geometric shapes as placeholders.
