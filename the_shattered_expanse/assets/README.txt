Place the following sprite files in this directory:

player.png - 32x32 pixels - Player character sprite
scout.png - 25x25 pixels - Fast enemy sprite
warrior.png - 32x32 pixels - Basic enemy sprite
heavy.png - 40x40 pixels - Tank enemy sprite
ranged.png - 28x28 pixels - Ranged enemy sprite
boss.png - 50x50 pixels - Boss enemy sprite

wall.png - 40x40 pixels - Basic wall structure
turret.png - 40x40 pixels - Advanced turret structure
storage.png - 60x60 pixels - Storage building
workshop.png - 80x80 pixels - Workshop building
collector.png - 40x40 pixels - Resource collector
generator.png - 50x50 pixels - Power generator

If sprite files are missing, the game will use colored shapes as placeholders.
